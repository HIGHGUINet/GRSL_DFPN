# GRSL_DFPN

### Requirements

### Usage
You can easily test the model.
1. Download [RSHaze weight](https://drive.google.com/file/d/1J6kgJE6G4s2X4841LKyr4Y9z3GO2worz/view?usp=drive_link) and make weight folder and put weight.
2. python base_test.py
